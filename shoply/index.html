Chrome & Edge Installation on Desktop

1

Open Shoply in Chrome/Edge

Navigate to the Shoply web app in your Chrome or Edge browser. Make sure you're on the main app page.

2

Look for the Install Icon

In the address bar, you'll see a small install icon ( or ). Click on it.

If you don't see the install icon, try refreshing the page or check if your browser supports PWAs.
3

Click "Install"

A popup will appear asking if you want to install Shoply. Click "Install" to proceed.

4

Launch the App

After installation, Shoply will open in its own window. You can now access it from your desktop or start menu.

Congratulations! Shoply is now installed as a PWA on your device.

Firefox Installation

1

Open Shoply in Firefox

Navigate to the Shoply web app in Firefox. Make sure you're on the main app page.

2

Access the Menu

Click the three-line menu button " " in the top right corner of Firefox.

3

Find "Install Site as App"

In the menu, look for "Install Site as App" or "Add to Home Screen" option and click it.

This feature may not be available in all Firefox versions. Update to the latest version if needed.
4

Complete Installation

Follow the prompts to complete the installation. Shoply will then be available as a standalone app.

Shoply is now an App on your device!

Safari (iOS) Installation

1

Open Shoply in Safari

Open Safari on your iPhone or iPad and navigate to the Shoply web app.

2

Tap the Share Button

Tap the share button " " at the bottom of the screen.

3

Select "Add to Home Screen"

Scroll down in the share menu and tap "Add to Home Screen".

4

Customize and Add

You can edit the app name if desired, then tap "Add" in the top right corner.

Shoply icon will now appear on your home screen like a native app!

Android Chrome Installation

1

Open Shoply in Chrome

Open Chrome on your Android device and navigate to the Shoply web app.

2

Look for the Install Banner

Chrome will automatically show an "Add to Home Screen" banner at the bottom. Tap it.

3

Alternative: Menu Method

If no banner appears, tap the three-dot menu " " in the top right, then select "Add to Home Screen".

4

Confirm Installation

Tap "Add" to confirm. Shoply will be installed and added to your home screen.

You can now launch Shoply like any other app from your home screen!

Quick Install Available

Most modern browsers will automatically prompt you to install Shoply when you visit the app. Look for install notifications!
